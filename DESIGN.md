# Design System

## Overview

Vibe uses a modern, clean design system built with Tailwind CSS. This document outlines the design principles, components, and styling guidelines.

## Color Palette

| Color | Usage |
|-------|-------|
| Primary Blue | Links, buttons, active states |
| Gray | Text, borders, backgrounds |
| Red | Error states, destructive actions |
| Green | Success states |

## Typography

- **Font Family**: System fonts for optimal performance
- **Font Sizes**: Responsive scale from text-sm to text-3xl
- **Font Weights**: Regular, medium, bold

## Components

### Post Card
- Image/video display with aspect ratio preservation
- Action buttons (like, comment)
- Caption and timestamp
- User avatar and name

### Navigation
- Clean top navigation bar
- User profile dropdown
- Responsive mobile menu

### Forms
- Consistent input styling
- Validation error states
- Loading states

## Layout

- Mobile-first responsive design
- Max width container for content
- Grid layout for post feed
- Flexbox for component alignment

## Icons

Using Heroicons for consistent iconography across the application.

## Accessibility

- Proper color contrast ratios
- Focus states for interactive elements
- Semantic HTML structure
- Alt text for images
